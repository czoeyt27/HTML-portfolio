# HTML Practice Projects

This repository contains two beginner web development projects created while learning HTML through a course on Udemy.

These projects were built to practice fundamental web development concepts such as page structure, headings, lists, links, and images.

## Projects Included

### 1. Birthday Invitation Page
A simple webpage designed to invite friends to a birthday celebration.

Features:
- Event title and description
- Date and event details
- Image integration
- List of items to bring
- Location link

### 2. Movie Ranking Page
A basic webpage listing and ranking favorite movies.

Features:
- Ordered and unordered lists
- Headings and paragraph elements
- Personal movie descriptions

## Technologies Used
- HTML5

## Purpose of the Project
These projects were created to practice core HTML concepts and build a foundation for future web development projects involving CSS, JavaScript, and interactive functionality.
